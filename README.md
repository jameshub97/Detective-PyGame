# Detective PyGame

**Game Jam Project :** A top-down detective RPG prototype built in 48 hours.

**Concept:** Play as a detective solving cases in a small town. Features tile-based movement, NPC interaction, and evidence collection.

**Tech:** Python, PyGame, custom sprites and tilemaps.

**What I learned:** 
- Game loop architecture and collision detection
- Managing multiple asset types (images, audio, tilemaps)
- Rapid prototyping under time constraints
- Structuring a real-time application

**Status:** Historical project from 2021. Demonstrates early exploration of Python and game development.
